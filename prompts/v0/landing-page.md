# Prompt v0.dev: Landing Page

Copia aquest prompt a v0.dev per generar una landing page de qualitat premium.

---

## Producte

**Nom del producte:** [NOM]
**Descripció breu:** [UNA FRASE]
**Públic objectiu:** [A QUI VA DIRIGIT]
**Problema que resol:** [PROBLEMA PRINCIPAL]
**Solució:** [COM HO RESOL]

## Estil visual

**To:** [Professional / Creador / Tècnic / Minimalista / Emocional]
**Colors principals:** [PALETA — no blau genèric]
**Tipografia:** [RECOMANACIÓ]
**Referències d'estil:** Stripe, Linear, Vercel (no copiar, usar com a guia de qualitat)

## Seccions de la landing page

1. **Hero** — Titular potent, subtítol, CTA principal, imatge o animació de fons
2. **Problema** — Explicació visual del problema que resol
3. **Solució** — Com el producte resol el problema (amb captures o il·lustracions)
4. **Beneficis** — Per què és millor que les alternatives (3-6 targetes)
5. **Funcionalitats** — Què fa el producte (detall visual, captures de pantalla)
6. **Prova social** — Testimonis, logos d'empreses, números d'ús
7. **Pricing** — Plans de preu clars amb CTA per pla
8. **FAQ** — Preguntes freqüents amb respostes
9. **CTA final** — Última oportunitat de conversió
10. **Footer** — Enllaços, legal, xarxes socials

## Storytelling

L'arc narratiu ha de ser:
1. **Aquest és el problema** (l'usuari s'hi identifica)
2. **Així és com se sent** (empatia)
3. **Aquesta és la solució** (el producte)
4. **Així és millor** (prova social, resultats)
5. **Fes això ara** (CTA)

## Restriccions tècniques

- React + Tailwind CSS + shadcn/ui
- Responsive (mòbil, tauleta, escriptori)
- Optimitzat per SEO (etiquetes semàntiques)
- Mode fosc/clar (si s'escau)
- Animacions subtils (scroll, hover) sense exagerar
- Càrrega ràpida (evitar imatges no optimitzades)

## Instrucció explícita

No generar una landing page genèrica tipus "hero + 3 caixes + footer".
Cada secció ha de tenir storytelling i personalitat pròpia.
Cuidar: microcopy, jerarquia visual, espai en blanc, ritme de lectura.
Fer servir shadcn/ui com a base però no limitar-s'hi.
